# Adani Safety

Adani Safety MIS + BSC dashboard prototype (HTML).

## Live site (GitHub Pages)

After [enabling GitHub Pages](#enable-github-pages), the dashboard is available at:

**[https://abhinandanunawane.github.io/Adani-Safety/](https://abhinandanunawane.github.io/Adani-Safety/)**

The root URL redirects to the app in [`prototype/`](prototype/).

### Enable GitHub Pages

1. Open the repo on GitHub: [abhinandanunawane/Adani-Safety](https://github.com/abhinandanunawane/Adani-Safety).
2. Go to **Settings** → **Pages** (left sidebar).
3. Under **Build and deployment** → **Source**, choose **Deploy from a branch**.
4. **Branch:** `main`, **Folder:** `/ (root)` → **Save**.
5. Wait 1–2 minutes, then open the **live link** above. Refresh if you still see 404 on the first try.

If you see “There isn’t a GitHub Pages site here”, Pages is not enabled yet or the build has not finished—complete the steps above and check the green success banner on the Pages settings screen.
